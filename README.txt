Тривиальная семейная книга, в которой мать вносит в базу данных задания, а отец назначает - кто будет выполнять данное задание. Так же отображается история сделанных проектов для всей семьи. В проекте сделанная простейшая авторизация пользователя - Имя и пароль. 
Технологии: PHP, ООР, Mysql, Bootstrap.
